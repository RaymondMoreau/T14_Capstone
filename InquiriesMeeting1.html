Talking Points for Project

What do we want to do for the wiki?

Consider using a text document or a dedicated wiki platform.

Points for Us:

  • Begin with design using tools like draw.io to structure.
  • Commit code regularly, even for small updates, week by week.
  • Maintain constant communication with the client and TA.
  • Discuss how content will be divided with the other group.

Capstone Questions:

  • How will we separate content?
  • Should content be separated by grade?
  • Which subjects are most important for interactive gamified learning?
  • Should analytics behind student grades be implemented?
  • Consider adding a messaging system for student-teacher communication.
  • Database for students' grades, activity, and times of activity?
  • Should teachers be able to add interactive content?
  • What level are we targeting? School board level or individual school level?
  • Consider color schemes, themes, and fonts for the website.
  • Host quizzes on the website and retain results?
  • What methods of testing functionality should be included?
  • Which games should be implemented?
  • Focus areas for subjects?
  • Consider a 'mini game library' for quests.
  • Design a teacher-student login system.
  • Decide on the process for creating student accounts.
